\questionhead{3}

\begin{itemize}
\item \begin{enumerate}
\item True, ($\Rightarrow$) if p is always true and q is also always true, then clearly at any time, both p and q are true together; same for $\Leftarrow$.
\item False, these are not equivalent, since p may be true at a different point in time than q, so that they are never true at the same time. (The $\Leftarrow$ part is true, since if they are eventually true together, than both of p and q are eventually true considered individually.)
\item True. p until q means that q will be true eventually, so $\Leftarrow$ holds; also for $\Rightarrow$, it doesn't matter when p is true or if ever, the LHS is true as soon as q becomes true.
\end{enumerate}
\item $f_1 U f_2$ becomes true if either $f_2$ is already true at that time, or $f_1$ is true and at the next step, the scenario is the same (either $f_2$ or $f_1$ and at the next step...)
\item $\lozenge\square\lnot(x>5)$
\end{itemize}
